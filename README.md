# 幽谷与丰筵 - 30天焦虑操练计划

## 文件说明

### 📁 countdown.html (倒计时页面)
**专门用于显示倒计时的页面**

✅ 包含功能：
- ⏰ 实时倒计时（天/时/分/秒）
- 💬 微信提醒按钮
- 🔒 美观的锁定界面
- 🔄 倒计时结束后自动跳转到主页面

**使用场景：**
- 3月1日之前访问，会自动跳转到这个页面
- 可以直接访问此文件查看倒计时效果

### 📁 index.html (主页面 - 完整版)
**包含完整的30天操练计划内容**

✅ 包含功能：
- 📅 30天完整操练计划
- 📖 每天的金句、书摘、操练、祷告
- 🧭 导航按钮和进度条
- ⏰ 访问时间检查（3月1日前显示倒计时）

**使用场景：**
- 3月1日之后访问，直接显示完整内容
- 包含所有30天的详细内容

### 📁 test-countdown.html (测试页面)
**简单的倒计时测试页面**

用于测试和调试倒计时功能。

## 如何使用

### 方案1：直接访问倒计时页面
```
访问：countdown.html
查看倒计时效果
```

### 方案2：使用主页面（带时间检查）
```
访问：index.html
3月1日前 → 自动显示倒计时
3月1日后 → 显示完整30天计划
```

## 部署建议

### Vercel / Netlify 部署
1. 上传整个目录
2. 设置默认页面为 `index.html`
3. 访问域名即可

### GitHub Pages
1. 推送到 GitHub
2. 在 Settings → Pages 中选择 main 分支
3. 访问 GitHub Pages URL

## 微信提醒功能说明

当前微信提醒功能为**前端演示版本**。

要实现真正的微信提醒，需要：
- 后端服务器
- 微信公众号/企业微信 API
- 用户授权系统
- 定时任务服务

**快速实现方案：**
使用第三方通知服务（如 Server酱、Bark 等）
